# GroceryTrack App

GroceryTrack is a simple and easy-to-use mobile application built with **React Native**. The app helps users manage and track their grocery items efficiently, making everyday shopping more organized and hassle-free.

## Overview

GroceryTrack allows users to keep a record of grocery items they need to buy or have already purchased. The app focuses on simplicity, clean UI, and smooth performance for a better mobile experience.

With GroceryTrack, users can:

* Add grocery items
* Update item details
* Remove items when no longer needed
* Keep their grocery list organized

## Features

* Add, edit, and delete grocery items
* Clean and minimal user interface
* Smooth mobile-friendly experience
* Lightweight and easy to use
* Built specifically for mobile devices

## Tech Stack

* **React Native**
* **JavaScript**
* **CSS / Styled Components (as used in project)**

## Running the Project Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/MsTanviAgrawal/GroceryTrack.git
   ```
2. Navigate into the project directory:

   ```bash
   cd GroceryTrack
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the app:

   ```bash
   npm start
   ```
5. Run the app on an emulator or physical device using Expo or React Native CLI
